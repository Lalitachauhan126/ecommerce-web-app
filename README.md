# E-Commerce Web Application

## 📌 Project Description
This is a simple E-Commerce Web Application developed using HTML, CSS, JavaScript, Node.js, and Express.js. The project allows users to browse products, add items to the cart, and access basic login, admin, and checkout pages.

## 🚀 Features
- Product Catalog
- Add to Cart
- User Login Page
- Admin Panel
- Shopping Cart
- Checkout Page
- Backend API using Express.js

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript
- Node.js
- Express.js

## 📂 Project Structure
```
ecommerce-web-app
│
├── public
├── routes
├── models
├── views
├── server.js
├── package.json
└── package-lock.json
```

## ▶️ How to Run
1. Install Node.js
2. Install dependencies:
   ```
   npm install
   ```
3. Start the server:
   ```
   npm run dev
   ```
4. Open in your browser:
   ```
   http://localhost:3000
   ```
